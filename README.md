# Heart Disease Prediction

## Project Overview
This project predicts whether a patient is likely to have heart disease based on medical attributes using machine learning classification techniques.

## Dataset
The dataset contains patient health information such as:
- Age
- Sex
- Cholesterol
- Blood Pressure
- Maximum Heart Rate
- Chest Pain Type
- Other clinical features

Target:
- 0 = No Heart Disease
- 1 = Heart Disease

## Data Preprocessing
- Handled missing values
- Treated outliers using capping
- Performed feature scaling
- Split data into training and testing sets

## Models Used
- Logistic Regression
- Decision Tree Classifier
- K-Nearest Neighbors (KNN)

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score

## Results
Logistic Regression produced the best overall performance with strong recall and balanced classification results, making it the most suitable model for this dataset.
